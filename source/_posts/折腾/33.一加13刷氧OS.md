## 软件下载

+ USB驱动
    + https://developer.android.com/studio/run/win-usb
+ 刷机包
    + https://sourceforge.net/projects/oneplus13flashers/files/Oneplus%2013/Super%20Flashers/
    + 当前使用的是欧版：SuperHybrid Flasher EU 15.0.0.864.zip
+ ROOT管理器
    + https://github.com/SukiSU-Ultra/SukiSU-Ultra/releases/tag/v4.0.0
+ ROOT插件
    + 修复国产应用与氧OS不兼容，导致运营商丢信号
        + https://github.com/K58/fix-signal-oneplus13/releases/tag/v3.2
        + 氧15使用3.2版本，氧16使用4.x


+ 启用 OEM 解锁
    + 设置 > 关于手机 > 版本，连续点击版本号7次
    + 设置 > 系统 > 开发者选项 > 启用 OEM 解锁
+ 启用 USB 调试
    + 在手机上，依次进入 “设置”> “ 关于手机” > “版本 ”，然后连续点击版本号 7 次以解锁开发者选项 。
    + 然后依次进入 “设置”> “ 系统和更新” > “开发者选项 ”，启用 “USB 调试” 。
+ 配置环境变量
    + `set PATH=%PATH%;\path\Platform-Tools`
+ 检查 ADB 是否正常工作
    + 执行`adb devices`，查询是否有列出设备
    + 可能会提示未授权，在手机里进行授权
+ 检查 Fastboot 模式
    + 执行`adb reboot bootloader`，进入 Fastboot 模式
    + 执行`fastboot devices`，查询是否有列出设备
    + 如果没有列出任何设备，在win的设备管理器里，未知Android设备。，右键更新驱动程序，选择USB驱动里的inf文件进行安装
+ 检查 Fastbootd 模式
    + 执行`fastboot reboot fastboot`，进入 Fastbootd 模式
    + 执行`fastboot devices`，查询是否有列出设备
+ 解bootloader锁
    + 执行`adb reboot bootloader`，进入 Fastboot 模式
    + 执行`fastboot devices`，查询是否有列出设备
    + 解锁并清除所有数据`fastboot flashing unlock`，手机里通过音量键选择解锁选择，电源键确认解锁，等待重启
    + 如果是要回锁的话，回锁并清除所有数据`fastboot flashing lock`（已ROOT状态无法回锁）
+ 启用 USB 调试
    + 设置 > 关于手机 > 版本，连续点击版本号7次
    + 设置 > 系统和更新 > 开发者选项
+ 刷入氧OS
    + 使用原装USB线连接，兼容性更好
    + 执行`adb reboot bootloader`，进入 Fastboot 模式
    + 执行`fastboot devices`，查询是否有列出设备
    + 运行`Super_Flasher.bat`文件
    + 中途应该会重启进入Recovery，选择`English`后回车继续
    + 期间会问你是否要root(y)，以及是否要清除数据(y)
    + Press any key to continue之后，在手机点击`Format data`，格式化数据，自动重启进入系统
    + 如果刷入失败或者无限重启
        + 长按电源+音量减，强制进入Fastboot
        + 执行`fastboot devices`，查询是否有列出设备
        + 如果没有列出任何设备，检查是否掉驱动了
        + 重刷
+ ROOT
    + 安装SukiSU`adb install SukiSU.apk`
    + 刷机包 > OOS_FILES_HERE > init_boot.img，复制到手机里，让SukiSU进行patch
    + 执行`adb reboot bootloader`，进入 Fastboot 模式
    + 执行`fastboot devices`，查询是否有列出设备
    + 刷入补丁`fastboot flash init_boot patch.img`
    + 重启手机`fastboot reboot`
    + 安装ROOT插件
